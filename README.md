# demo-getafe-notaria-sagrario-alvarez

Demo conceptual no oficial para Notaria Sagrario Alvarez.

Sector: notaria
Ciudad: Getafe
Fuente: https://www.notariasagrarioalvarez.com/
Contacto publico: msalvarez@correonotarial.org
Precio propuesto: 720 EUR

Punto debil trabajado:
La web tiene buena base, pero los textos de cookies en ingles y la jerarquia simple restan claridad en tramites de alto valor.

Stack: HTML estatico, CSS embebido y vercel.json para despliegue en Vercel.
